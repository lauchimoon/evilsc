# evilsc
Screenshot program

## Getting started
```
$ git clone https://github.com/lauchimoon/evilsc.git
$ cd evilsc/
$ make
$ ./evilsc -o [file] -d [delay]
```
Both `[file]` and `[delay]` parameters are optional.

If `[file]` parameter is not specified, it defaults to `out.png`

`[delay]` parameter is in seconds.
